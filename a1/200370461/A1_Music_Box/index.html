Assignment #1: Music Box

Instructions:

Create an HTML Text Box that allows user input of coded "notes"; a "note" can be any single character you choose, it can be range from "A,B,C,D,E,F,G". Next, a number in-front of the note for how many times it should play in a row. Lastly, add a decimal in-front of the note for how fast the note should play.

Button (Interface):

  • Clear - It will empty the text from the box
  • Run - It will run the Music Box
  • Clear Note - It will clear the Music Note Notation from Reference and Real-Time
